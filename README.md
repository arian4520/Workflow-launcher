# Workflow-launcher

## Description
A Python-based GUI application that allows users to create and run workflows consisting of URLs and file paths.

## Features
- Add and delete URLs/files
- Run all items with one click
- Save multiple workflows
- Load saved workflows
- User-friendly interface

## Technologies Used
- Python
- Tkinter
- webbrowser module
- os module

## How to Run
1. Install Python
2. Run `main.py`
3. Use the interface to create workflows
